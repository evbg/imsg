# imsg

**imsg** - parser of telegraph messages in iata and icao formats

The code is extracted from the old project and will be refactored.

### Requirements

imsg has no third-party dependencies other than the standard Python library.


## Running the tests

```
python ./test_iata.py
python ./test_icao.py
```
